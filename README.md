# Splitwiz 💸

A trip expense-splitting web app that makes shared spending simple. Create a trip, invite your group with a join code, and track who paid for what, no more messy spreadsheets or manual math after a trip.

Live app: https://expense-splitter-two-flax.vercel.app

## Features

Create a trip and generate a shareable join code for your group. Add and track shared expenses as they happen. Automatic calculation of who owes whom. Bank import and payment tracking. Email and push notifications for updates. Analytics and profile dashboard. Progressive Web App (PWA) support.

## Tech Stack

Framework: Next.js. Language: TypeScript. Database: Turso (SQLite) with Prisma ORM. Styling: Tailwind CSS. Backend: Supabase. Deployment: Vercel.

## Getting Started

Clone the repo and install dependencies, then run the dev server.

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser to see the app running locally.
